# Movie Trailer Website Project

This is a python generated website featuring some of my **FAVE** movies past, present and future.

** Interested in making one for yourself. Keep reading.

## Requirements

<ul>
    <li>Install <a href="www.python.com">Python</a> </li>
</ul>

## Directions

<ol>
    <li>Fork this <a href="https://github.com/udacity/ud036_StarterCode"> Repository</a> for starter files. </li>
    <li> Create data structure to store **FAVE** movie info (i.e. movie title, box art URL and Youtube video).</li>
    <li> Create multiple instances of that Python Class to showcase each movie</li>
    <li> Reference the starter files forked from repository above</li>
    <li> Ensure website renders correctly.</li>
</ol>
